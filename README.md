# Design System Example App: vue v3  + tailwind v4

This repo is designed to act as living documentation/playground/guide as to how to integrate the Design System into different libraries/frameworks.

### Getting Started

```shell
git clone git@github.com:anyvan/design-system-example-vue-3-tailwind-4.git
cd design-system-example-vue-3-tailwind-4
# authentication for CodeArtifact
aws sso login --profile horizonta
./codeartifact.sh
npm i
npm run dev
```
